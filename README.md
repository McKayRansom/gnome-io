# Gnome-IO

A 2d colony-sim inspired by Gnomoria but designed to be simpler — survive the seasons
and goblin raids with a colony of gnomes. See [DESIGN.md](DESIGN.md) for the full pitch.

Notes are split across three files:

- **[ROADMAP.md](ROADMAP.md)** — what's in flight and what's next (SIDEVIEW-WIP → MVP → Kingdom).
- **[DESIGN.md](DESIGN.md)** — the design philosophy, feature goals, Gnomoria comparisons, and open questions.
- **[ICEBOX.md](ICEBOX.md)** — speculative faction/creature ideas, parked until the core loop is playable.
